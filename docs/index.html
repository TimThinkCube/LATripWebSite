美國洛杉磯之旅

Plan a real journey rather than trash talk

The first trip we planned by our own.

Ever since, we’ve been thinking how to make it true together.

Now Just Do It!

Here’s how we got started…

故事是這樣開始的，我們是一群大學期間都會混在一起吃飯、打屁、玩樂、讀書？的朋友，比較特別的是有一位尼加拉瓜的好夥伴Ｊohnny，雖然是一個外國人， 但是是一個有趣的外國人，一天到晚 幹您娘，幹您娘，不明所以的喊著，還記得我們當時一起選修測量學這門課，然而大部分的同學都不知道 如何跟一個歪國人溝通，而我呢也只是一個不知道測量在幹什麼的高中生，因此我們這兩個狀況外的人，意外成為了好朋友，哈哈。我開始透過簡單的翻譯和肢體互動，和這個 老外說明課堂上所謂的實習要做什麼？老師再幹麻，我想我們的友情就是從互相幫忙開始建立起來，漸漸的每次的活動提醒一下Ｊohnny已經變成不只是我，也是我這些慢慢 混在一起的夥伴們會一起做的事情。

時間過得很快，這位歪國人除了長得帥一點以外，在生活上也帶給我們很多的樂趣及啟示。因此在畢業前，我們就說好，等到我們有了工作能力，有了穩定的收入到時候一定要去尼加拉瓜 找你玩，雖然到時候小強尼可能已經跑出來了，哈哈。時至今日我們從畢業規劃這件事情到執行也已經默默過了5年，現在我們真的要去找你囉，Ｊohnny，不過碰面的地方不是在 尼加拉瓜大瀑布也不是尼加拉瓜，而是在夢之城 Ｌos Angeles(洛杉磯)
